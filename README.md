# project-todo-list

A simple project-global todo list that can be accessed with a keyboard shortcut. The extension creates one todo file in the extension's storage per workspace/project.

## Features

Press `cmd+shift+j` or use the `project-todo-list.show-todo-list`-command to access the todo list.

## Requirements

None.

## Extension Settings

None, this extension is simple. Use the regular keybindings editor to change keybindings.
